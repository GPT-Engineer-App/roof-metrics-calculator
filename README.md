# roof-metrics-calculator

Aerial roof and measurement website to enable users to upload photos of roofs and automatically calculate measurements and other details needed for roofing estimates. The site will provide an interface for uploading images, and display calculated metrics such as area, perimeter, and material requirements.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/roof-metrics-calculator.git
cd roof-metrics-calculator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
